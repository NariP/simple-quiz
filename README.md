# Simple Quiz

### 설치
> nvm use
> 
> npm i

### 실행
> npm run dev

### 안내
```agsl
백엔드 서버가 꺼져있을 수도 있습니다.
.env 파일 없으면 로컬 실행 안됩니다.
실행 전 꼭 nvm 으로 노드 버전을 맞추고 .env 파일을 루트에 추가해주세요.

api 이슈로 인해 로컬환경에서만 실행됩니다;;
```
